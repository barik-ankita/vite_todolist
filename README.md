# TodoList App

This project is a simple TodoList application created using Vite, React, Tailwind CSS, and React Icons. It allows users to add, edit, and delete tasks, providing a streamlined way to manage their todos.


## Features

- **Add Tasks**: Users can add new tasks to the list by typing in the input field and pressing enter.
- **Edit Tasks**: Users can edit existing tasks by clicking on the task text, making it easy to update task details.
- **Delete Tasks**: Users can delete tasks individually by clicking on the delete icon associated with each task.
- **Mark Tasks as Complete**: Users can mark tasks as complete by clicking on the checkbox icon, providing visual feedback on task completion.
- **Responsive Design**: The app is designed to be responsive, ensuring optimal usability across various devices such as desktops, tablets, and mobile phones.
- **Styled with Tailwind CSS**: Utilizes Tailwind CSS for styling, allowing for rapid development and customization.
- **Icons from React Icons**: Uses React Icons library for easily adding icons to the user interface.
- **Built with Vite**: Developed using Vite for fast and efficient build times during development.
